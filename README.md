# Construction Company Website

A modern, responsive commercial website built with [Next.js](https://nextjs.org) for a construction company.  
This website showcases services, past projects, and contact information, designed with SEO optimization and performance in mind.

---

## About the Company
This website was developed for **MirzoBuild**, a professional construction company specializing in residential, commercial, and industrial projects.  
The goal was to provide potential clients with an easy way to learn about the company's services, view past projects, and get in touch.

---

## Features
- **Modern UI/UX** – Built with React & Next.js for a smooth browsing experience
- **Fully Responsive** – Optimized for mobile, tablet, and desktop
- **SEO Friendly** – Meta tags, optimized images, and fast load times
- **Service & Project Showcase** – Highlight construction expertise
- **Contact Form Integration** – Easy communication with potential clients

---

## Tech Stack
- [Next.js](https://nextjs.org) – React framework for server-side rendering and static site generation
- [React](https://reactjs.org) – UI component library
- TailwindCSS – Styling
- [Vercel](https://vercel.com) – Deployment and hosting

---

## Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/yourusername/construction-website.git
cd construction-website
```

### 2️⃣ Install Dependencies
```bash
npm install
# or
yarn install
```

### 3️⃣ Run the Development Server
```bash
npm run dev
# or
yarn dev
```

### Build for Production
```bash
npm run build
npm start
```

### Deployment
```bash
The easiest way to deploy is via Vercel:
Push your code to GitHub
Go to Vercel and import your repository
Deploy in one click
```
